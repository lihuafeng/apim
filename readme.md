## 小手API性能监控系统 apim.xs25.cn

小手Api性能监控系统,基于elasticsearch中的NGIN请求日志,分析请求时间,访问量,错误码等.
- 及时发现响应慢,报错接口
- 自由接入项目接口
- 接口使用频率统计
- 自动报警,可定制化报警
- 汇报统计（优化后效果）
- 历史数据永久保留
